# Skills

Store reusable AI/developer skills in this folder.

Examples:

- prompt templates
- review checklists
- coding standards

## Included Skills

- [code-using-tdd/SKILL.md](code-using-tdd/SKILL.md): Guides implementation using strict red-green-refactor cycles, with example prompts for GitHub Copilot Chat.

## How To Use Skills

Open the workspace in VS Code and refer to the skill by name in Copilot Chat. Example prompts:

```text
Use the code-using-tdd skill for this implementation.
```

```text
Apply code-using-tdd and start with a failing test.
```
